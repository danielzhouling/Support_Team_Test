# Dmall Helpdesk 技能评估系统 - 设计规范

## 1. 设计理念

**关键词**：专业、高效、清晰、现代

**设计风格**：企业级后台系统风格，参考 Ant Design / Element Plus 的设计语言，简洁但不简单。

**目标用户场景**：
- 管理端：Admin 在办公室 PC 使用，需要处理大量数据
- 员工端：Helpdesk 员工可能在门店移动端使用，需要快速完成考核

```

### 字号规范
```

H2 (模块标题):    20px / 600 weight

Body (正文):      14px / 400 weight
Small (辅助文字):  12px / 400 weight
Tiny (标签):      10px / 400 weight

## 10. 组件规范

### 按钮
```
主要按钮: 背景#409EFF, 白色文字, hover变深
次要按钮: 白色背景, #409EFF边框和文字
文字按钮: 无边框, #409EFF文字
危险按钮: #F56C6C背景

尺寸:
- 大: 40px高, 16px字号
- 中: 32px高, 14px字号 (默认)
- 小: 24px高, 12px字号
```

### 输入框
```
高度: 32px (中), 40px (大)
边框: 1px solid #DCDFE6
聚焦: 边框变#409EFF
圆角: 4px
```

### 表格
```
表头: #F5F7FA背景, 14px/600
单元格: 白色背景, 14px/400
行高: 48px
Hover: #F5F7FA背景
```

### 卡片
```
背景: 白色
圆角: 8px
阴影: 0 2px 12px rgba(0, 0, 0, 0.1)
内边距: 20px
```

### 弹窗
```
圆角: 12px
阴影: 0 8px 24px rgba(0, 0, 0, 0.15)
遮罩: rgba(0, 0, 0, 0.5)
```

## 11. 响应式断点

```
PC:  >= 1200px  (最优体验)
Tablet: 768px - 1199px (适配平板)
Mobile: < 768px (手机端)

管理端: 主要针对PC，最小宽度1024px
员工端: 移动端优先，适配手机屏幕
```

## 12. 图标规范

使用 Element Plus 内置图标，辅以 Font Awesome 或 IconFont。

常用图标：
- 账号: el-icon-user
- 题库: el-icon-document
- 考核: el-icon-edit-outline
- 报表: el-icon-data-analysis
- 设置: el-icon-setting
- 通知: el-icon-bell
- 搜索: el-icon-search
