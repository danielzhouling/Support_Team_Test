# 需求变更记录

## 2026-04-23 - 项目初始化

### 背景
Dmall 菲律宾 SM 项目 Helpdesk 团队需要一个「AI辅助出题 + 在线考核 + 智能分析」一体化的技能评估解决方案。

### 原始需求
- 产品：技能评估系统
- 目标用户：Helpdesk 团队（11人）+ 管理员（1-3人）
- 核心功能：账号管理、文档管理、题库管理、考核管理、数据报表、员工端、AI能力

### 决策
1. **分三阶段开发**：MVP → AI增强 → 数据学习
2. **新增阶段0**：在开发前先做静态HTML Demo，确认UI/UX方案后再coding
3. **技术栈**：Vue3 + FastAPI + SQLite + Minimax 2.7
4. **MVP范围**：阶段一只做核心考核闭环，AI能力放阶段二

### 研发计划

| 阶段 | 内容 | 周期 |
|------|------|------|
| 阶段0 | 静态HTML Demo（16个页面） | 1-2周 |
| 阶段一 | MVP核心闭环 | 2-3周 |
| 阶段二 | AI能力增强 | 2-3周 |
| 阶段三 | 数据与学习 | 1-2周 |

### 备注
- 具体coding时间以实际为准
- 每阶段需可验证
- 开发前必须完成所有功能的静态Demo页面
